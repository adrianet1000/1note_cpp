Setup Clang on Windows and VsCode without Microsoft Visual Studio IDE
==================================================

go to https://www.msys2.org or just search for msys2

1) Download and install mysys2
2) Open mysys2 terminal and type the the following to install the clang package:

pacman -S mingw-w64-x86_64-clang
pacman -S mingw-w64-x86_64-clang-tools-extra


3) add Clang path to the System variable : path

C:\msys64\mingw64\bin

en esta dirección se encuentra el compilador clang++.exe

luego para compilar un archivo 

clang++ test.cpp -o test.exe

para ejecutar el test.exe

clang++ .\test   en powershell
test             en command cmd de windows10

COMPILAR

Compilar es pasar un fichero fuente (codigo fuente) a codigo maquina. El codigo fuente puede estar escrito en c++, java, c# y muchos otros lenguajes. Es decir un ordenador no entiende otra cosa que no sea codigo maquina, es decir codigo binario "unos y ceros.", el codigo fuente es mas cercano al idioma con el que nos comunicamos las personas (Español, Ingles, etc.), y el codigo maquina es el lenguaje mas cercano para que fuesen interpretados por las maquinas.

Lenguages de alto nivel

Porque su sintaxis, su escritura se asemaja mucho al idioma que hablan los seres humanos mientras se les denomina lenguajes de bajo nivel a los lenguajes que se aproximan al lenguaje maquina como puede ser el lenguaje ensamblador.

cuatro pasos que se sigue al compilar

Preprocesado
compilación
ensamblado
enlazado

El preprocesado es ver todas las directivas en el caso de un archivo de c++ para poder importar todo el codigó que hay en esas librerias, para que podamos usar todas esas clases y funciones que hay en esas librerias.

En la segunda etapa tambien llamada compilación por coincidencia, es la etapa donde se generan instrucciones de ensamblado que van a depender de la maquina donde se este compilando el programa, es decir que si estas utilizando una maquina de 32bits se generara un codigo de ensamblado determinado, pero si estas utilizando una maquina de 64bits se generara un código de ensamblado distinto.

El tercer paso que se denomina ensamblado, esas instrucciones de ensamblado del paso dos, se transforma a codigo binario

En el último paso se fusionan todos los archivos que se hayan podido generar. En un unico archivo ejecutable, en nuestro caso sera un archivo.exe que finalmente sera el archivo que se lee, se ejecuta, se interpreta

¿Qué es una directiva?

Es una instrucción que nos permite añadir contenido que hay en bibliotecas externas a nuestro programa

Espacios de nombres

Los espacios de nombres estan dentro de las bibliotecas por lo que podemos precindir de los espacios de nombres en el archivo c++ pero la función cout ne sería reconocido directamente por el compilador de c++. por lo que tendriamos que enteponer el espacio de nombre lugo doble dos puntos para luego poner la instrucción cout como se muestra en el siguiente ejemplo.

#include <iostream>

int main(){
	std::cout << "Hola mundo!" << std::endl;
	return 0;
}

Se puede notar que la función endl tambien se encuentra dentro del espacio de nombres std

Declarar una variable

Es la acción de crear una variable, sin todavía darle un valor

Inicializar una variable

Es darle valor a una variable, es decir almacenar un tipo de dato dentro de la variabla ya creada previamente.

int salario = 3500; //Declarar e iniciar una variable
int ventas;  //Declarar una variable 
    ventas = 6000; //Inizializar una variable."Notar que no se necesita nuevamente int previamente."

Inicializar una variable en c++

int z;      //Sin inicializar.
int z = 5; //Inicialización tradicional.
int z(5);  //Inicialización de constructor.
int z{5};  //Inicialización de lista.

int z{5}; esta forma de inicializar esta variable se creo recientemente y que permite resolver perdida de presición al momento de realizar la conversión de datos implicita conocida como narrowing.

Ejemplo de inicialización de variables

#include <iostream>
using namespace std;

int main(){
    string nombre = "Adrian";
    int edad{35};
    double salario;
    cout << "Introduce el salario del empleado:\n";
    cin >> salario;
    cout << "Datos del empleado: \n";
    cout << "Nombre: " << nombre << ". Edad: " << edad << ". Salario: " << salario;
    return 0;
}

	\n \t \b etc

	Palabra reservada const

	Palabra reservada constexpr
Constantes enumeradas:
	Palabra reservada enum

	Directiva #define (obsoleto)

Un ejemplo de las constantes definidas es:

#define precio = 1200

//Notar que no se necesito decirle el tipo de dato y tampoco
//se le puso el punto y coma al final de la instrucción.

Ejemplo de la declaración e inicialización de constantes.

#include <iostream>
using namespace std;

int main(){
    const string nombre = "Adrian";
    int edad{35};
    double salario;
    cout << "Introduce el salario del empleado:\n";
    cin >> salario;
    cout << "Datos del empleado: \n";
    cout << "Nombre: " << nombre << ". Edad: " << edad << ". Salario: " << salario;
    return 0;
}

Notar que no se permite declarar e inicializar la constante en lineas distintas, es decir si la constante que queremos crear la inicializamos en otra linea, el compilador de c++ nos genera un error.
